A simple plugin that forces Wordpress default search feature to search only posts and excludes pages from the search results.

== Description ==

A simple plugin that forces Wordpress default search feature to search only posts and excludes pages from the search results.

== Installation ==

On Wordpress Dashboard click on Add New Plugin, and search for "Search Only Posts", or download the plugin and:

1. On your dashboard go to Plugins and click on 'Add New'.
2. Click on 'Upload Plugin', search the plugin on your computer and upload it.
3. Activate the plugin.
